# CryptoClass - Trading Masterclass

## Overview
CryptoClass is a modern, responsive e-learning platform designed to deliver structured cryptocurrency trading education. The platform features a progressive curriculum spanning beginner to pro levels, complete with progress tracking and interactive lesson modules.

## Features
* **Structured Curriculum:** Organized into distinct phases (Week 1: Beginner, Week 2: Intermediate, Week 3: Advanced, Week 4: Pro, plus Bonus Sessions).
* **Progress Tracking:** Visual indicators showing course completion percentage.
* **Interactive Lessons:** Expandable accordion modules for clean content consumption.
* **Theme Support:** Built-in Light/Dark mode toggle.
* **Responsive Design:** Optimized layout for seamless learning across desktop and mobile devices.
